# Contribuindo

Obrigado por contribuir com o Radar do Povo.

## Diretrizes

- mantenha a estrutura padrão por dataset;
- documente a fonte oficial do dado;
- preserve nomenclatura consistente;
- implemente imports idempotentes;
- adicione migrations e documentação do dataset;
- não introduza acoplamento desnecessário entre datasets.

## Commits

Sugestão de convenção:

- `feat(dataset): ...`
- `fix(dataset): ...`
- `docs: ...`
- `chore: ...`

## Pull Requests

Todo PR deve:

- explicar o dataset afetado;
- informar a fonte oficial;
- listar mudanças em schema ou importação;
- atualizar documentação se necessário.
